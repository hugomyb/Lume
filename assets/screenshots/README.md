# Screenshots

Drop the README screenshots/GIFs here, then uncomment the `<table>` block in the
root `README.md` (under "## Screenshots").

Suggested set (PNG or GIF, ~1280px wide, dark theme):

| File          | What to show                                                       |
| ------------- | ----------------------------------------------------------------- |
| `blocks.png`  | A few command blocks with exit-code badges + the blocks sidebar.  |
| `ai.png`      | The AI palette or an inline "explain block" response.             |
| `panes.png`   | A split layout (2–3 panes) on a nice theme.                       |
| `remote.png`  | The remote page on a phone (tab bar + key row), or the QR dialog. |

Tips:
- Use a clean prompt and a couple of real commands.
- A short GIF for the AI/remote ones sells them better than a still.
- Keep file sizes reasonable (compress with e.g. `oxipng` / `gifsicle`).
